## $${\color{red}Color \space formats}$$

| Format | Explication | Description | Syntax |
|------|----------|----|---------|
| | | |
| Name | Color Name | Use predefined color names | black |
| Hex | Hexadecimal | Define colors using six-digit hex codes | #FE6932 |
| RGB | Red, Green, Blue) | Define colors using RGB values | rgb(123, 0, 69) |
| RGBA | Red, Green, Blue, Alpha | RGB extended by alpha (transparency) | rgba(123, 0, 69, 0.4) |
| HSL | Hue, Saturation, Lightness | Define colors using HSL | hsl(100, 69%, 50%) |
| HSLA | Hue, Saturation, Lightness, Alpha | RGB extended by alpha (transparency) | hsla(123, 100%, 69%, 0.6) |
| OKLCH | Oklab - Lightness, Chroma, Hue / Alpha | Define colors using LCH/A values | oklch(20% 0.7 140 / 0.6) |
