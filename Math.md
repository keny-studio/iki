Common Functions
abs(num): Returns the absolute (positive) value.
pow(base, exponent): Raises a base to a power.
sqrt(num): Returns the square root.
max(n1, n2, ...): Returns the largest value.
min(n1, n2, ...): Returns the smallest value.
pi(): Returns the value of Pi (approx. 3.14159).
rand(min, max): Generates a random integer. 

The round() function rounds a floating-point number. Tip: To round a number UP to the nearest integer, look at the ceil() function. Tip: To round a number DOWN to the nearest integer, look at the floor() function.
